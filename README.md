# Prerequisites
- node.js
  1. Install node.js
  1. `sudo npm install -g http-server`
- Or, See [here](https://github.com/processing/p5.js/wiki/Local-server)

# Local Server
1. run local server
    ```
    cd ${the_directory_of_this_repository}
    http-server
    ```
1. Launch your browser and accsess to "http://127.0.0.1:8080"
